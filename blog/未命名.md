### 1. Clone the Repository

[](https://github.com/lehome-official/lehome-challenge/blob/main/docs/installation.md#1-clone-the-repository)

```shell
git clone https://github.com/lehome-official/lehome-challenge.git
cd lehome-challenge
```

### 2. Install Dependencies with uv

[](https://github.com/lehome-official/lehome-challenge/blob/main/docs/installation.md#2-install-dependencies-with-uv)

```shell
uv sync
```

This will create a virtual environment and install all required dependencies.

### 3. Clone and Configure IsaacLab

[](https://github.com/lehome-official/lehome-challenge/blob/main/docs/installation.md#3-clone-and-configure-isaaclab)

```shell
cd third_party
git clone https://github.com/lehome-official/IsaacLab.git
cd ..
```

### 4. Install IsaacLab

[](https://github.com/lehome-official/lehome-challenge/blob/main/docs/installation.md#4-install-isaaclab)

Activate the virtual environment and install IsaacLab:

```shell
source .venv/bin/activate
./third_party/IsaacLab/isaaclab.sh -i none
```

### 5. Install LeHome Package

[](https://github.com/lehome-official/lehome-challenge/blob/main/docs/installation.md#5-install-lehome-package)

Finally, install the LeHome package in development mode:

```shell
uv pip install -e ./source/lehome
```

---

If you are using a server, please download the system dependencies.

```shell
    #step 1
    apt update
    apt install -y \
    libglu1-mesa \
    libgl1 \
    libegl1 \
    libxrandr2 \
    libxinerama1 \
    libxcursor1 \
    libxi6 \
    libxext6 \
    libx11-6
    #step 2
    export __GLX_VENDOR_LIBRARY_NAME=nvidia
```


### 2. Assets & Data Preparation

[](https://github.com/lehome-official/lehome-challenge?tab=readme-ov-file#2-assets--data-preparation)

#### Download Simulation Assets

[](https://github.com/lehome-official/lehome-challenge?tab=readme-ov-file#download-simulation-assets)

Download the required simulation assets (scenes, objects, robots) from HuggingFace:

```shell
# This creates the Assets/ directory with all required simulation resources
hf download lehome/asset_challenge --repo-type dataset --local-dir Assets
```

#### Download Example Dataset

[](https://github.com/lehome-official/lehome-challenge?tab=readme-ov-file#download-example-dataset)

We provide demonstrations for four types of garments. Download from HuggingFace:

```shell
hf download lehome/dataset_challenge_merged --repo-type dataset --local-dir Datasets/example
```
